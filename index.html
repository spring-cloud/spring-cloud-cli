Here's a complete Eureka Server: ```groovy @EnableEurekaServer @RestController class Eureka {} ``` To install the CLI, make sure you have [Spring Boot CLI](https://github.com/spring-projects/spring-boot) (1.2.0 or better): $ spring version Spring CLI v1.2.1.RELEASE E.g. for GVM users ``` $ gvm install springboot 1.2.1.RELEASE $ gvm use springboot 1.2.1.RELEASE ``` and install the Spring Cloud plugin: ``` $ mvn install $ spring install org.springframework.cloud:spring-cloud-cli:1.0.0.BUILD-SNAPSHOT ``` IMPORTANT: **Prerequisites:** to use the encryption and decryption features you need the full-strength JCE installed in your JVM (it's not there by default).
